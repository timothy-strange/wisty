# Wisty v2 (Scaffold)

This repository now hosts the TypeScript rewrite scaffold for Wisty using Tauri + SolidJS.

- Legacy implementation is archived in `legacy-reference/`.
- Architecture and rewrite guidance are documented in `IMPROVEMENT_PLAN.md`.
- Both `legacy-reference/` and `IMPROVEMENT_PLAN.md` are ignored in git and will not be pushed.

## Development

- `npm install`
- `npm run dev`
- `npm run tauri dev`

## Build

- `npm run build`
- `npm run tauri build`
